# 🚗 Indian License Plate Detection

This project detects Indian vehicle number plates using YOLOv8 and extracts text using Tesseract OCR.

## 🔥 Features
- License Plate Detection (YOLOv8)
- OCR Text Extraction (Tesseract)
- Image & Video Support
- Streamlit UI

## 🛠️ Tech Stack
- Python
- Streamlit
- OpenCV
- Ultralytics YOLOv8
- Tesseract OCR
